#vimfiles

##Vim Customizations

###Installation

```
git clone https://github.com/mbasanta/vimfiles.git ~/.vim
```

###Create symlinks:

```
ln -s ~/.vim/vimrc ~/.vimrc
ln -s ~/.vim/gvimrc ~/.gvimrc
```

###Set up Vundle

Clone Vundle

```
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

Open Vim and run `:PluginInstall`
